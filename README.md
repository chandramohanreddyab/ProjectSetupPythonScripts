## Usage

Basic usage (create a folder `MyApp` and scaffold a Flask app inside it):

```bash
python foldersetup_flask_app.py /full/path/to/MyApp
```

Examples:

- Create a new project folder and scaffold files:

```bash
python foldersetup_flask_app.py /full/path/to/MyApp
```

- Force creation inside an existing non-empty folder (use with care):

```bash
python foldersetup_flask_app.py /full/path/to/MyApp --force
```

Notes:

- `project_path` is a required positional argument and should be the full path to the project folder to create.
- The script will exit with an error if the target folder exists and is not empty, unless `--force` is provided.
- The generated scaffold creates a `tests/` folder by default; no separate flag is required for tests.

Run `python foldersetup_flask_app.py --help` for exact flags supported by your local copy of the script.
